## Features

- **Particle network background** with mouse interaction
- **Code rain background** (Matrix-style falling characters)
- **Custom cursor** with hover effects + **magnetic buttons**
- **3D morphing object** - click to morph between 5 shapes, drag to rotate
- **Scroll progress indicator** - gradient bar showing page position
- **Text scramble effect** - headings decode when entering viewport
- **Mobile hamburger menu** - responsive navigation
- **Elegant minimal footer** - Dostoevsky quote, clean CTA
- **Scroll reveal animations** with Intersection Observer
- **Contact form** (uses Web3Forms - add your key)
- **Konami code easter egg** (↑↑↓↓←→←→BA)
- **Keyboard accessible** with focus states
- **Reduced motion support** for accessibility

## 🚀 Quick Start (Local)

```bash
cd "sharanu's portfolio"
npx serve .
# or
python3 -m http.server 8000
```

Then open `http://localhost:3000` (or 8000)

## 🌐 Deploy to Vercel

### Option 1: Vercel CLI (Fastest)

```bash
# 1. Install Vercel CLI
npm i -g vercel

# 2. Navigate to portfolio folder
cd "sharanu's portfolio"

# 3. Deploy
vercel

# Follow prompts → get live URL
```

### Option 2: Drag & Drop

1. Go to https://vercel.com/dashboard
2. Drag the entire `sharanu's portfolio` folder onto the dashboard
3. Done!

## 📝 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
  --bg: #080b14;          /* Background */
  --surface: #0d1220;     /* Surface/Card bg */
  --orange: #ff6b2b;      /* Main accent */
  --orange-light: #ff8f5e; /* Light accent */
  --text: #e8eaf0;        /* Primary text */
  --muted: #7a8099;       /* Muted text */
}
```
